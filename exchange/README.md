# exchange

A small skill-exchange project (Node.js) containing backend APIs and frontend code for managing users, skills, and skill exchange requests.

## Project Structure

- `backend/` — Node.js backend modules (user, skill, requests, admin/demo scripts)
- `frontend/` — Frontend application (place your UI here)

## Getting started

1. Install dependencies

```bash
# if there's a package.json
npm install
```

2. Run backend

```bash
# Run individual scripts (example)
node backend/admin.js
node backend/demo.js
```

3. Frontend

Place or build your frontend into the `frontend/` folder and serve using your preferred method.
